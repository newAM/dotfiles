## Setup

### Spellcheck
```
:set spell spelllang=en_ca
:set spellfile=~/.vim/spell/en.utf-8.add
```

* `z=` alternative spelling
* `zg` add to dictionary
* `zw` mark incorrect

Temporarily disable:
```
:set nospell
```
